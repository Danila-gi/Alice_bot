# Бот алисы

## Описание
Данный проект содержит исполняемые файлы для вывода работы бота и запуска тестов.

## Загрузка

### Загрузка GTests
1. Перейти в корень проекта
2. Склонировать репозиторий: ```git clone https://github.com/google/googletest.git```

### Сборка проекта
1. Создать папку сборки и перейти туда: ```mkdir build && cd build```
2. Собрать проект: ```cmake .. && make```
3. В папке build появятся 2 исполняемых файла: ```alice_bot``` - для запуска приложения, ```alice_bot_tests``` - для запуска тестов

## Использование
Для запуска приложения надо находиться в папке build (можно использовать тестовые файлы, можно создать свои): ```./alice_bot ../inputFiles/in2.txt```

Для запуска тестов надо находиться в папке build: ```./alice_bot_tests```
